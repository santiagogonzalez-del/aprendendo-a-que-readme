É o título principal do README e representa o nome do sistema ou trabalho.

Explique o que o sistema faz e qual é o objetivo dele.
Exemplo:
Sistema para cadastro de alunos em uma escola.
Status do Projeto
Mostra se o projeto:
ainda está sendo desenvolvido
já foi finalizado.
